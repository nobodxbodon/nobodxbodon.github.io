安装Zsh

我在mac下，使用port install zsh就好了，因为mac自带的Zsh比较老，然后使用chsh -s /opt/local/bin/zsh搞定

安装Zsh配置文件

克隆配置

git clone git://github.com/sjl/oh-my-zsh.git ~/.oh-my-zsh

移动到根目录

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
