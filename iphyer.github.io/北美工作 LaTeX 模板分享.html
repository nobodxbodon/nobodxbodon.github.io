Overleaf

当然现在已经不推荐大家自己安装 LaTeX 了，虽然是很锻炼人的一个过程但是太不舒服了。至今记得当年就着校园网的蜗牛网速下载 TeXLive 最新版的过程。 现在都是推荐 Overleaf。这网站已经是 CVPR 这些 CS Top Conference 推荐的编辑器了。同时可以使用谷歌等拼写检查工作特别适合写作。

当然也很方便多人合作，共同努力。可以说是效率神奇，上个学期 CS 744 用过之后再也不喜欢别的编辑器了。

当然上面提供的下载链接也可以直接把 zip 文件上传到 Overleaf 就可以新建一个项目了，非常方便。
