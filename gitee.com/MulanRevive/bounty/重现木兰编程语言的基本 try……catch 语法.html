木兰的 try...catch 基本语法

可以 catch 所有错误，不用指定具体类型：

try {
  print([][0])
} catch e {
  print(e)
}

也可指定错误类型：

try {
  print(int('x'))
} catch e : ValueError {
  print('v')
}

catch 多个类型错误：

try {
  print([][int('0')])
} catch e : ValueError {
  print('v')
} catch e : IndexError {
  print('i')
}

值得注意的是，从 Python 代码改写为木兰代码时，容易写出这样的代码：

try {
  print([][int('0')])
} catch ValueError {
  print('v')
} catch IndexError {
  print('i')
}

此时会报错：default 'except:' must be last
