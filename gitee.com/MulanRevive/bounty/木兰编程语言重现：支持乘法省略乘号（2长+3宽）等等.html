乘法可省乘号

一个数和某些表达式相乘时，支持省略乘号。这个设计应该是为了接近数学算式的表示。

此语法 Python 并不支持。

长 = 2
宽 = 3
print(3长 + 4 宽)

语义为3 个长加上 4 个宽，输出 18。

支持继承两个类

type Animal {
    func getName() {
        return 1
    }
}
type 当地 {
{}
}
type Person : Animal {
{}
}
type 当地人 : Person, 当地 {
{}
}

支持多项 for 迭代

for x, y in zip(['1', '2'], ['a', 'b']) {
  print(x + y)
}

输出1a2b。
