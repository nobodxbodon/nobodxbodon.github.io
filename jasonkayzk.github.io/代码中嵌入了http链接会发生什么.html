C语言代码中嵌入http链接会发生什么

猜一下，下面的C++代码能正常编译嘛：

#include <iostream>

int main() {
    using namespace std;

    https://jasonkayzk.github.io/
    cout << "hello, world" << endl;
    return 0;
}

答案是可以正常编译，并输出hello, world；

原因其实很简单：

https被当作了一个标签，而后面的//被当作了注释！
