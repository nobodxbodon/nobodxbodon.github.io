原贴地址：<a href='http://spaces.msn.com/members/conge/Blog/cns!B6DC33938CD14B1!425.entry'>CongeLoveFreya</a> 作者：Conge 使用<a href='http://mianwo.blogspot.com/2007/03/blogsync.html' title='Blog同步工具'>BlogSync™</a>复制<br/><br/>从<a href="http://calon.weblogs.us/">滯銷書</a>那里看到，<a href="http://technorati.com/">Technorati</a> 好像已经被封杀了，就过去看了看。我是没有使用那里的服务的。因为自己的收心阁还没有成什么气候，当然，我也没有想让它成什么气候，不过这样的网站说不定还是会用的。当然，吸引我的不是<a href="http://technorati.com/">Technorati</a> 提供什么，而是它的被封杀。<br><br>当初在北大使用教育网的时候，就总为访问国外站点头痛，也因此练就了一身寻找使用代理的本事。那时，一听到国内的大学喊“建设世界一流大学”的口号时候心里就想骂，nnd我们想看看什么世界一流大学的网站都困难重重，还谈什么建设！扯吧...<br><br>西方国家的人民，尤其是美国人民，对我们的国家有许多的偏见，有些偏见确实也是达到了非常夸张的地步，不过我们的确没有真正意义上的 言论自由。 路透社的<a href="http://www.google.com/search?hl=en&amp;lr=&amp;rls=GGGL,GGGL:2006-10,GGGL:en&amp;q=%E5%8D%9A%E5%AE%A2%E9%97%A8&amp;btnG=Search">博客门</a>事件固然有其记者失掉职业操守的一面，然而王三表这么容易让西方媒体着了道儿的原因，也是因为他利用了我们自己国家自己的病。<br>我不懂政治上的东西。也不能谈。我只是想自由的使用互联网的各种服务，当然是在国内的时候。我希望什么都能进入自己的耳朵和眼睛，然后自己再做出选择，听什么，看什么。<br><br>下面是<a href="http://calon.weblogs.us/">滯銷書</a>的全文<br><p>数一数哪些曾经用过的优质服务和站点被封杀了：</p><br /><p>最可惜的是<a href="http://wikipedia.org/">维基百科</a><br><br />电信网络再次无法访问的 <a href="http://www.weblogs.us/">Weblogs.us</a><br><br />这一次的 <a href="http://technorati.com/">Technorati</a><br><br />曾经是首选的网摘服务 <a href="http://www.furl.net/">Furl</a><br><br /><a href="http://www.google.com/">Google</a> 的 blog 服务 <a href="http://www.blogger.com/">Blogger.com</a> 、网页快照以及一些搜索结果<br><br /><a href="http://wordpress.org/">WordPress</a> 的 blog 服务 <a href="http://wordpress.com/">WordPress.com</a><br><br />免费站点服务 <a href="http://geocities.yahoo.com/">Yahoo! GeoCities</a><br><br /><a href="http://www.xys.org/">新语丝</a><br><br />因为安替而遭殃的 <a href="http://www.blog-city.com/">Blog-City</a></p><br /><p>曾经被封杀，但不定哪一天又会进去的：<br><br /><a href="http://www.blogsome.com/">BlogSome</a><br><br />开源软件站点 <a href="http://www.sourceforge.net/">SourceForge<br></a><a href="http://www.freebsd.org/">FreeBSD</a><br><br />在线电子书籍项目 <a href="http://www.gutenberg.org/">Project Gutenberg</a></p><br /><p>当然，还有很多服务、软件站点，以及数不清的 BSP/Blog 没有列进来，我想能够提供完整名单的不在我这里，以后有权利向公安部和政府要求公开被封杀站点的时候，再看看那壮观的列表吧。</p><br /><p>我想问的是，哪一个服务被封杀了会让你觉得最愤怒呢？哪一个是最不可缺少的呢？</p><br /><p>在已经被封杀的服务中，应该是<a href="http://wikipedia.org/">维基百科</a>吧，还健在的服务中，也许是 <a href="http://del.icio.us/">Del.icio.us</a> 和 <a href="http://mail.google.com/mail">Gmail</a>。</p><br /><p>许多人担心会被封杀的站点：</p><br /><p>社会性书签服务 <a href="http://del.icio.us/">Del.icio.us</a><br><br />处理 Feed 的 <a href="http://feedburner.com/">FeedBurner</a><br><br />Feed 订阅的 <a href="http://www.bloglines.com/">Bloglines</a><br><br />相册存储与分享站点 <a href="http://www.flickr.com/">Flickr</a><br><br />新闻组 <a href="http://groups.google.com/">Google Groups</a></p><br /><p><a href="../">MSN Spaces</a> 之流就不用担心会被封杀了，因为它首先会把用户出卖掉。<br /></p>
