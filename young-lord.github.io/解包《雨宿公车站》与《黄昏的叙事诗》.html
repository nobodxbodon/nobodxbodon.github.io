附录

一个将繁中资源文件转换为简中的代码：

from zhconv import convert
for i in os.listdir():
    with open(i,"rb") as f:
        ff=f.read().decode("utf8")
    with open(i,"w",encoding="utf8") as f:
        f.write(convert(ff,'zh-cn'))
