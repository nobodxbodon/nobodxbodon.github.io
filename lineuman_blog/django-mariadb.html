django + mariadb

创建项目django-admin startproject your_project

settings.py

DATABASES = {
    'default': {
            'ENGINE': 'django.db.backends.mysql',
            'NAME': 'cve',
            'USER': 'root', 
            'PASSWORD': 'your_password', 
            'HOST': 'localhost',
            'PORT': '',
            }       
}

 ALLOWED_HOSTS = [ipaddress]

注意只有运行了python manager.py migrate之后相关的数据库才会被创建
