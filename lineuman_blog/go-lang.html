go-lang

install

yum install go-lang

example

helloworld.go

package main
import "fmt"
func main(){
        for true{
                fmt.Println("helloworld")
        }

}

go build helloworld.go;./helloworld

or

go run helloworld.go

写在最后

随着经验的增长，越来越意识到，哪种语言或者哪种语法并不重要，重要的是够利用计算机的能力去解决问题，当然不同的语言有不同的生态环境，选择语言势必要考虑其背后的生态，究竟是选择一门全新的语言，还是选择有着浓重历史包袱与积淀的语言，这就仁者见仁智者见智了。
