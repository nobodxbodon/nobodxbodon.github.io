<p>测试代码：</p>

<?php
    #echo print 函数效率测试对比
    $old = microtime(true);

    for($i = 0; $i < 10000000; $i++){
        echo "1";
    }

    $new = microtime(true);
    $one = $old - $new;
    echo $one . "\n"; #echo 函数测试结果

    $old = microtime(true);

    for($i = 0; $i < 10000000; $i++){
        print "1";
    }

    $new = microtime(true);
    $two = $old - $new;
    echo $two . "\n"; #print 函数测试结果

    echo ($two - $one) / $two; #效率比较
?>

<p>测试结果：</p>

$one:  0.35927200317383
$two:  0.42829704284668
($two - $one) / $two:  0.16116160694007

<p>多次测试，发现echo比print效率高15%左右。</p>

<h3><strong>结果分析</strong></h3> <p>echo和print功能上差不多，但是在基层的定义中：</p>

int print(argument);
void echo(string argument1, [, ...string argumentN]);
