仓库源文，站点原文

先在这里直接下载编辑器代码。 插入 javascript 代码如下，

<script charset="utf-8" src="kindeditor/kindeditor.js"></script>
<script charset="utf-8" src="kindeditor/lang/zh_CN.js"></script>
<script>
        var editor;
        KindEditor.ready(function(K) {
                //editor = K.create('textarea[name="content"]');
                editor = K.create('#editor_id', {
                    allowFileManager : true  //开启浏览服务器文件功能，比方上传图片的图片空间
                });
        });
</script>

form 表单代码如下：

<textarea id="editor_id" name="content" style="width:90%;height:450px;"></textarea>

Kindeditor 内容区默认字体和字体的大小如何设置呢？ 打开 themes/default/default.css 文件找到：.ke-dialog-loading-content 把font-size: 14px;改成font-size: 12px;保存即可。
