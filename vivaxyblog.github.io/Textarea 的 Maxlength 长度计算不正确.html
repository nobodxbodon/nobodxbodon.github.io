layout: post title: Textarea 的 Maxlength 长度计算不正确

tags: [html, textarea]

Chrome 下的 textarea 的长度限制后，如果其中包括换行的话，能输入的字符会少于 maxlength 中限定的长度。

<textarea
    placeholder="请输入驳回原因"
    maxlength="10"
    >
</textarea>

测试结果

浏览器 回车占几个字符
chrome 46.0.2490.86 (64-bit) 2
chrome 46.0.2490.86 m 2
ie 11 1
edge 1
firefox 42.0 1
