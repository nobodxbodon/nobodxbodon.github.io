原文：本地静态资源服务器

layout: post title: 本地静态资源服务器 tags: [NPM, JavaScript, Tool, Server]

适用于前端开发的静态资源服务器。

  • 基于 nodejs。
  • 支持 get 和 post 请求。
  • 端口被占用时，自动换端口启动服务。
  • 支持 livereload，here -w 开启，监听 html，js，css 文件变化，并自动刷新。

安装方式：

sudo npm install -g serve-here
