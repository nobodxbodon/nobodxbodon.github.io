layout: post title: Git Commit Script

tags: [zsh, git]

When using git, I always typed following command in order:

git add .
git commit -m 'foobar'
git push

Then I installed zsh, I typed:

ga .
gcmsg 'foobar'
gp

Now I wrote a plugin for zsh: gcmt, I just typed:

gcmt
enter commit message: foobar

It's all done!
